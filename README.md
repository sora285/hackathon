# my-vue-apli

## 初回セットアップ
```
npm install
```

### 開発環境の起動　開発を開始するには、次のコマンドを実行します。
```
npm run serve
```

### 本番ビルド 本番環境用にアプリをデプロイする準備をします
```
npm run build
```

### コードの確認と修正 プッシュする前にコードをチェックします。
```
npm run lint
```
